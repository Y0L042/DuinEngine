Detailed Description

template<typename... types>
class duin::Signal< types >

Signal class for event-driven programming.

A type-safe signal/slot system supporting multiple listeners. Listeners are identified by UUIDs and can be connected, disconnected, and notified with arbitrary arguments. NOT THREAD-SAFE!

Move-only: moving a Signal preserves all existing ScopedConnections.

Template Parameters
typesArgument types passed to listeners when the signal is emitted.
duin::Signal<int, std::string> sig;
duin::UUID id = sig.Connect([](int a, const std::string& b) { ... });
sig.Emit(42, "hello");
sig.Disconnect(id);
