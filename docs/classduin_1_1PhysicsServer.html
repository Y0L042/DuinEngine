Detailed Description

Singleton managing the PhysX physics world.

Handles PhysX initialization, scene management, and simulation stepping. Physics bodies (StaticBody, DynamicBody, etc.) use this internally.

Connect to PhysX Visual Debugger (PVD) for real-time debugging.
