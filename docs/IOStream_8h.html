Detailed Description

C++ RAII wrapper around SDL3's SDL_IOStream API.

Provides a move-only, type-safe stream I/O class with automatic resource cleanup. Supports file, memory, dynamic memory, and custom streams, as well as endian-aware typed reads/writes and bulk file operations.

See also
https://wiki.libsdl.org/SDL3/CategoryIOStream
