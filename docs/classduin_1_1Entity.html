template<typename Relation >
Entity & duin::Entity::Add ( const Entity &  target)
inline

Add a relationship pair to the entity. This creates a (Relation, target) pair where Relation is a component type and target is another entity. This is commonly used for relationships like Likes, Targets, Owns, etc.

Example:

Entity player = world.CreateEntity("Player");
Entity enemy = world.CreateEntity("Enemy");
player.Add<Targets>(enemy); // Player targets enemy
