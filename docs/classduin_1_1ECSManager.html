Detailed Description

Central ECS world manager.

Owns the FLECS world and provides methods for entity creation, component registration, and query execution. Each scene/tab in the editor has its own ECSManager instance.

ECSManager ecs;
ecs.Initialize();
duin::Entity e = ecs.world.CreateEntity()
.IsA(ECSPrefab::Node3D)
.Set<ECSComponent::Transform3D>({position});
