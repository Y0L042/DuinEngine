Detailed Description

Wrapper for RapidJSON Document/Value, providing type-safe JSON manipulation.

JSONValue allows parsing, writing, and modifying JSON data, supporting objects, arrays, and primitive types. It manages memory using shared_ptr for the underlying RapidJSON Document.
