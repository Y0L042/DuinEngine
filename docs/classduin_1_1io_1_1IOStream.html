Detailed Description

RAII, move-only wrapper around SDL3's SDL_IOStream.

Owns an SDL_IOStream pointer and automatically closes it on destruction. Use static factory methods to create instances. Copy is deleted; move transfers ownership.

See also
https://wiki.libsdl.org/SDL3/CategoryIOStream
