Detailed Description

Stack-based state manager.

Manages a stack of GameState objects. The top state receives lifecycle callbacks. States can be pushed (pausing current), popped, or switched.

stateMachine.PushState<MenuState>();
// Later...
stateMachine.SwitchState<GameplayState>();
