Detailed Description

Internal wrapper structure for bridging custom callbacks to SDL callbacks.

This structure holds both the user's callback function and their userdata pointer, allowing AdapterDirCallback() to properly forward calls to the user's callback while translating between Duin's EnumerationResult and SDL's SDL_EnumerationResult.

Note
This is an implementation detail and should not be used directly
