Detailed Description

Defines the duin::JSONValue class for manipulating JSON data using RapidJSON.

Author
DuinEngine Contributors
Date
2024

This file provides the JSONValue class, a wrapper around RapidJSON's Document and Value types, enabling convenient and type-safe manipulation of JSON objects, arrays, and primitive values. It also provides iterator classes for traversing JSON arrays and objects.

Usage example:

duin::JSONValue json = duin::JSONValue::Parse("{\"key\": 42}");
int value = json["key"].GetInt();
