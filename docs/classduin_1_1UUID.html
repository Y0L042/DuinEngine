Detailed Description

A 64-bit universally unique identifier.

UUID provides a lightweight unique identifier system based on 64-bit integers. It supports random generation, conversion to/from string formats (decimal and hexadecimal), and hashing for use in standard containers like std::unordered_map.

Thread-safety note: UUID generation uses a shared random engine. While the generation itself is not thread-safe due to the shared engine state, each generated UUID is unique with extremely high probability.
