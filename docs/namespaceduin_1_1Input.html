Types of key/button events.

Enumerator
PRESSED 

Key was just pressed this frame.

PRESSED_REPEATED 

Key is repeating (held down, OS repeat).

HELD 

Key is currently held down.

RELEASED 

Key was just released this frame.

IDLE 

Key is not pressed.
