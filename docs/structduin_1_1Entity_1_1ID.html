Detailed Description

Wrapper around flecs::id_t that provides entity ID manipulation. A flecs id is an identifier that can be added to entities. Ids can be:

  • entities (including components, tags)
  • pair ids
  • entities with id flags set (like flecs::AUTO_OVERRIDE, flecs::TOGGLE)
