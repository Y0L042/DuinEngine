Detailed Description

Virtual-path wrappers around duin::io::IOStream file operations.

Provides free functions in the duin::vio namespace that resolve virtual path prefixes (bin://, app://, usr://, wrk://) to system paths via fs::MapVirtualToSystemPath, then delegate to the corresponding duin::io::IOStream static methods.
