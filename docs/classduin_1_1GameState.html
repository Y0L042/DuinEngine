Detailed Description

Base class for application states.

Represents a single state in the state machine (e.g., menu, gameplay). Override lifecycle methods to implement state behavior. Each state has its own GameObject hierarchy via stateGameObject.

class MenuState : public duin::GameState {
public:
MenuState(GameStateMachine& owner) : GameState(owner) {}
void Enter() override { // Setup menu UI }
void Update(double delta) override { // Handle menu input }
void Exit() override { // Cleanup }
};
