Detailed Description

Action-based input system.

Provides a layer of abstraction over raw input. Named actions can be bound to multiple input sources (keyboard keys, mouse buttons) and queried by name rather than device-specific codes.

// Setup
duin::CreateInputAction("jump");
duin::AddInputActionBinding("jump", keyboard, DN_KEY_SPACE, Input::KeyEvent::KEY_PRESSED);
// Query
if (duin::IsInputActionTriggered("jump")) {
// Handle jump
}
