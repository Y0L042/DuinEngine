Detailed Description

GameObject hierarchy and state machine.

The Objects module provides a traditional scene graph alongside the ECS. GameObjects form parent-child hierarchies and receive lifecycle callbacks. GameStateMachine manages application states (menu, gameplay, pause, etc.).
